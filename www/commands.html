Übersicht der Befehle
Befehl Aliases Bedeutung
/quit /exit, /raus, /logoff, /bye Den Chat verlassen. Alternativ kann man auch auf "Logout" klicken.
/nick Nickname   Der eigenen Nicknamen verändern.
/color Farbe /farbe, /col Ändert die aktuelle Farbe oder gibt die aktuelle Farbe aus, falls keine Farbe angegeben. Die Farbe muß entweder ein RGB-Wert sein oder eine Bezeichnung wie z.B. rot. RGB-Werte lassen sich am einfachsten mit der Farbpalette (unter Profile / Profil bearbeiten) erzeugen.
/room Raumname /raum, /join, /channel, /go In den Raum Raumname wechseln.
/msg Nickname Text /talk, /tell, /t Privatnachricht an den Benutzer schicken. Falls der Name nicht vollständig ist, wird er automatisch ergänzt. Privatnachrichten können von niemandem außer den Adressaten gelesen werden (auch nicht von Administratoren).
/me Text /txt Einen Spruch in den Raum schreiben.
/away Text /weg In den Abwesenheitsmodus umschalten, eventuell mit einem Begründungstext. Der Anwesenheitsmodus wird beendet, sobald der Benutzer etwas schreibt.
/smileys on bzw. /smileys off /smileyson bzw. /smileysoff Smileys an- und ausschalten. Solange die Smileys aus sind, werden die Smileys in den Texten anderer Chatter nicht in Bilder umgewandelt. Außerdem wird die Smiley-Auswahl nicht angezeigt.
/ignore nick   Einen Benutzer ignorieren, es werden dann keine Nachrichten mehr von diesem Benutzer angezeigt. Falls kein Nickname angegeben, wird die Liste der ignorierten Benutzer ausgegeben. Die Liste der ignorierten Benutzer wird bei jedem Einloggen gelöscht.
/unignore nick   Die Ignorierung eines Benutzers aufheben, so daß wieder Nachrichten von ihm empfangen werden können.
/invite nick /einlad Nur für Raumbesitzer und Administratoren: Einen Benutzer in einen geschlossenen Raum einladen. Der Benutzer erhält eine Nachricht und kann den Raum betreten. Die Einladung ist dauerhaft, sie wird nur durch /uninvite aufgehoben. Falls kein Nickname angegeben, wird die Liste der eingeladenen Benutzer ausgegeben.
/uninvite nick /auslad Nur für Raumbesitzer und Administratoren: Die Einladung eines Benutzers aufheben. Der Benutzer bleibt in dem Raum, falls er schon dort ist, wenn er ihn aber verläßt, kann er nicht wieder zurückkommen.
/addvip nick   Nur für Moderatoren: Einen VIP für den aktuellen Raum ernennen, dessen Texte dann von allen im Raum gelesen werden können. Im Gegensatz zu einem Moderator liest ein VIP jedoch nicht die Texte aller Benutzer, sondern nur die von Moderatoren und anderen VIPs. Die Ernennung ist dauerhaft, sie wird nur durch /removevip aufgehoben. Falls kein Nickname angegeben, wird die Liste der VIPs in diesem Raum ausgegeben.
/removevip nick   Nur für Moderatoren: Den VIP-Status eines Benutzers aufheben.
/img URL Nickname   Nur für Administratoren: Das durch die URL angegebene Bild an einen Benutzer schicken. Falls kein Nickname angegeben, wird das Bild an jeden im Raum geschickt.
/broadcast Text /atall, /analle Nur für Administratoren: Einen Text an alle Benutzer im Chat schicken.
/kick Nickname   Nur für Administratoren: Einen Benutzer aus dem Chat schmeißen.
/push Nickname Raum /schubs Nur für Administratoren: Einen Benutzer in einen Raum schubsen. Falls der Name nicht vollständig ist, wird er automatisch ergänzt.
/gag Nickname Zeit /knebel Nur für Administratoren: Einen Benutzer für eine gewisse Zeit zum Schweigen bringen. Die Zeit wird in Minuten angegeben, falls nicht angegeben werden 5 Minuten genommen. Der Befehl wird aufgehoben, indem man als Zeit 0 angibt. Falls der Name nicht vollständig ist, wird er automatisch ergänzt.
